## 📌 주요 기능

### 🌐 글로벌 리스크 게이지
- 0~100 리스크 지수 실시간 표시
- 30일 히스토리 차트 (Chart.js)
- 리스크 요인 태그 자동 업데이트

### 📊 KOSPI 수급동향
- 외국인·기관·개인 순매수 현황
- 수급 막대 그래프 시각화
- 수급 데이터 직접 입력 및 저장

### 📰 경제·정세 뉴스
- AI 자동 요약 뉴스 카드 5개
- 카테고리 태그 및 포트폴리오 영향도 표시
- 🔄 업데이트 버튼으로 실시간 갱신

### 💼 자산배분 권고
- 5대 자산군 비중 카드 (국내주식·미국주식·금·원유·현금)
- 시나리오 3가지 전환
  - ⚔️ 전쟁 장기화
  - 🕊️ 단기 종결
  - 📉 글로벌 침체

### ⚡ 섹터별 분석
- 국내주식 / 미국주식 탭 전환
- 섹터 위험도 바 차트
- 섹터 클릭 시 상세 분석 팝업

### 🤖 AI 리스크 어드바이저
- Claude API 연동 실시간 투자 상담
- 포트폴리오 입력 시 맞춤 분석
- 빠른질문 5개 버튼
- API 키 없이도 기본 오프라인 답변 제공

## 🚀 사용 방법

### 1. 기본 사용 (API 키 없이)
파일을 열면 바로 사용 가능해요.
샘플 데이터로 모든 기능을 확인할 수 있어요.

### 2. AI 기능 활성화 (Claude API 키)
```
1. console.anthropic.com 접속
2. API Keys → Create Key
3. 대시보드 하단 AI 어드바이저 → API 키 입력
4. 🔄 업데이트 버튼으로 실시간 데이터 수신
```

> 💡 API 키는 브라우저 로컬 스토리지에만 저장되며 외부로 전송되지 않아요.
